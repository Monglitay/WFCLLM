*阶段三重构版**：

### 阶段三：提取与验证阶段 —— 基于 SEMSTAMP 经验阈值的 DP 检验

**核心目标：** 将 LSH 区域落点判定映射为 SEMSTAMP 的“有效句子/代码块”统计，利用 DP 提取独立的最大不重叠代码块集合，并严格按照经验假阳性率（FPR）确定最终的判定边界。

#### 1. 结构解析与目标重构（保持不变）

解析待测代码的 AST。针对每一个语句块 $i$，利用其父节点上下文与全局安全密钥，重新计算出生成时该位置对应的“有效区集合 $G_i$”。

#### 2. LSH 语义特征打分与有效性判定（适配 SEMSTAMP 计数逻辑）

提取代码块的语义向量 $u_i$，计算其 LSH 签名 $c_i$。

**打分规则变更：** SEMSTAMP 的核心是统计落在有效区域内的单元数量 。因此，我们需要将原本的 $\{-1, 1\}$ 计分改为 $\{0, 1\}$ 计分：

* 若 $c_i \in G_i$，说明代码块落入**有效区域**，记为命中（$S_i = 1$）。
* 若 $c_i \notin G_i$，说明代码块落入**受阻区域**，记为未命中（$S_i = 0$）。

#### 3. 动态规划（DP）去重与独立样本提取

继续使用你的 DP 方程，由于状态值变为 $\{0, 1\}$，该方程现在的物理意义是：**寻找 AST 树中，能产生最大有效命中数的、互不嵌套的独立代码块集合**。


$$OPT(i) = \max \left( S_i, \sum_{j \in \text{children}(i)} OPT(j) \right)$$

回溯 DP 状态数组，提取出这批互不包含的独立语句块。

* 设提取出的**独立代码块总数**为 $S_T$。
* 设其中**落入有效区域的数量**为 $S_V$。

#### 4. SEMSTAMP 的单比例 Z 检验与 FPR 动态阈值（核心替换）

这里我们将引入 SEMSTAMP 论文中基于假阳性率的动态阈值定位机制。

**确立零假设 $H_0$：**
该文本的生成（或编写）并不知道语义空间中有效和受阻分区的规则 。

**计算 SEMSTAMP Z 得分：**
设有效区域占总区域的比例为 $\gamma$。使用 SEMSTAMP 的公式计算得分：


$$z_{\text{SEMSTAMP}} = \frac{S_V - \gamma S_T}{\sqrt{\gamma(1-\gamma)S_T}}$$



**动态假阳性率（FPR）判定阈值校准：**
放弃硬编码的 $Z > 3.0$。按照 SEMSTAMP 的机制，当 $z_{\text{SEMSTAMP}} \ge M_r$ 时，我们检测出机器生成的文本，其中 $M_r$ 是根据给定的假阳性率 $r$ 定位的 。
在系统部署前，你需要在无水印的自然代码语料库（负样本集）上运行上述 1-3 步：

* 遍历可能的阈值范围 $m \in [0, 6.0]$ 。


* 找到一个特定的 $M_r = m$，使得恰好有 $r$ 百分比（例如 1% 或 5% ）的自然代码被误分类为机器生成文本 。



**最终判定：**
在实际检测时，若计算出的 $z_{\text{SEMSTAMP}} \ge M_r$，则拒绝零假设，确认为注入了水印的代码 。
