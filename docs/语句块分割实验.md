# 语句块分割实验
数据集：MBPP
目标：将代码块分割为独立的语句块
工具：使用AST解析器 tree-sitter
使用python 使用conda WFCLLM环境 下载需要的包
# 使用方法：
1. 安装tree-sitter库
2. 下载MBPP数据集
from datasets import load_dataset
ds = load_dataset("google-research-datasets/mbpp", "full")
3. 遍历数据集中的每个代码块
4. 使用tree-sitter解析代码，提取语句块
语句块分简单语句块和复合语句块，简单语句块是不嵌套其他语句块的语句块，复杂
块是嵌套其他语句块的语句块
1. 整个数据程序文件和结果文件等实验放在experiment目录下单独建立一个文件夹，数据集放在 experiment外面一层目录叫做data/mbpp
