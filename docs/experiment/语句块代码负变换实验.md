# 语句块代码变换实验
# 目的：编写AST变换规则，对语句块进行变换
# 实验方法：
1. 使用AST解析器tree-sitter，对experiment/statement_block_split/results/mbpp_blocks.json 中的语句块进行语义等价变换
2. 语义变换的规则遵循docs/python负代码变换规则.csv
3. 每个语句块检测所有可行的代码变换
4. 每个语句块对所有可行的规则进行全部的排列组合，然后进行变换
5. 每个变换完了的语句块进行保存，，标注好使用了那些规则，以及属于那个源代码（id)的负语义变换
# 要求：
1. 代码逻辑，api接口的格式尽量和experiment/statement_block_transform一样，这样方便后序移植
2. 流程和experiment/statement_block_transform一样，只是其中应用的规则变了，变成构造语义负样本的。
