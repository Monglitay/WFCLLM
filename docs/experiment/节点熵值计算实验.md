# 节点熵值计算实验

# 目标：
我需要计算的是 AST 节点的综合信息熵。假设节点 $N$ 包含 $k$ 个 Token，第 $i$ 个 Token 在给定前缀 $p_i$ 时的概率分布为 $P(v|p_i)$，请严格按照以下公式聚合：
$$H_{\text{Node}}(N) = \frac{1}{k} \sum_{i=1}^{k} \left( -\sum_{v \in C(p_i)} P(v|p_i) \log_2 P(v|p_i) \right)$$

# 实验步骤：
0. **数据来源**：experiment/statement_block_transform/results 中源代码和AST变换后的带代码
1. **数据准备**：从 AST 中提取出所有节点及其包含的 Token。
2. **概率计算**：对于每个节点，计算每个 Token 在不同前缀下的条件概率分布。
3. **熵值计算**：根据公式计算每个节点的综合信息熵。
4. **结果分析**：对计算得到的熵值进行统计分析
